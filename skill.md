Non-Negotiables

These requirements are mandatory and must not be ignored, simplified, or replaced during design or development.



Brand Positioning

Propertycare Paints & Electrics is a premium property transformation company.

The website must communicate:

* Quality craftsmanship
* Professionalism
* Reliability
* Luxury finishes
* Property transformation expertise

The website must NOT feel like:

* A handyman website
* A budget contractor website
* A generic tradesman template
* A directory-style business listing
* A low-cost service provider

Visitors should immediately feel they are dealing with professionals who deliver high-quality property transformations.

⸻

Mobile-First Design

The website must be designed mobile-first.

Most visitors will discover the business from:

* Google Search
* Google Business Profile
* WhatsApp
* Facebook
* Instagram

The mobile experience is the primary experience.

Desktop layouts should be adapted from mobile designs, not the reverse.

⸻

Contact Strategy

The primary objective of the website is generating enquiries.

Contact hierarchy:

1. WhatsApp
2. SMS Text Message
3. Phone Call
4. Contact Form (Optional)

The user should never need to search for contact information.

Every page must contain:

* WhatsApp CTA
* SMS CTA
* Phone CTA

⸻

Floating Contact Actions

Persistent floating actions are required:

* WhatsApp Button
* Text Message Button
* Call Button

These actions must remain visible throughout the user journey.

⸻

Hero Section Requirements

The hero section must:

* Be visually impressive
* Communicate trust instantly
* Establish premium positioning
* Showcase transformation work

The hero section must NOT:

* Use stock template layouts
* Feel generic
* Be text-heavy
* Look like a local contractor website

A cinematic visual approach is preferred.

⸻

Portfolio First

The portfolio is the strongest sales tool.

Project showcases must be prioritized above lengthy service descriptions.

Show:

* Before & After Comparisons
* Transformation Stories
* Media Wall Projects
* Electrical Installations
* Painting Projects
* Property Improvements

Whenever possible:

Show proof before making claims.

⸻

Imagery Standards

Only use:

* Real client projects
* Professional photography
* Premium AI-generated placeholders

Never use:

* Generic stock construction photos
* Low-quality images
* Pixelated images
* Irrelevant trade imagery

The imagery should reinforce premium positioning.

⸻

Service Pages

Service pages must focus on outcomes rather than tasks.

Bad Example:

“We install sockets and switches.”

Good Example:

“Modern electrical upgrades that improve safety, functionality, and appearance.”

Sell results.

Not labour.

⸻

Animation Standards

Animations should feel premium and refined.

Allowed:

* Fade reveals
* Staggered content reveals
* Smooth image transitions
* Elegant hover effects
* Subtle motion

Not Allowed:

* Excessive parallax
* Bouncing animations
* Distracting effects
* Overly aggressive motion

Performance always takes priority.

⸻

Typography Rules

Large, confident typography.

Headings should create impact.

Spacing should feel luxurious.

Avoid:

* Small unreadable text
* Dense layouts
* Tight spacing

The website should feel premium and spacious.

⸻

Conversion Optimisation

Every section should answer one question:

“Why should I trust this company with my property?”

Every major section should include a logical conversion path.

Examples:

Portfolio Section:
→ Request Similar Transformation

Media Wall Section:
→ Get a Media Wall Quote

Painting Section:
→ Request Painting Estimate

Electrical Section:
→ Speak With an Electrician

⸻

SEO Standards

Every location page must provide genuine value.

Never create thin SEO pages.

Location pages should include:

* Service information
* Area-specific content
* Testimonials
* Project examples
* Contact options

Target areas include:

* Sunderland
* Washington
* Seaham
* South Shields
* Gateshead
* Newcastle upon Tyne
* Durham
* Chester-le-Street
* Jarrow
* Hebburn
* Peterlee

⸻

Accessibility Standards

Must meet modern accessibility requirements.

Requirements:

* Keyboard navigation
* Proper semantic HTML
* ARIA labels where necessary
* Colour contrast compliance
* Accessible forms

Accessibility must not be treated as optional.

⸻

Performance Standards

Minimum Lighthouse Scores:

Performance: 95+

Accessibility: 95+

Best Practices: 100

SEO: 100

The website must feel fast, responsive, and polished on all devices.

⸻

Final Quality Test

Before completion, evaluate the website using this question:

“Would a homeowner trust this company with a £20,000+ property improvement project after viewing this website?”

If the answer is not an immediate yes, continue improving the design, content, imagery, UX, and presentation until it reaches that standard.

The website should feel comparable to premium property transformation, interior renovation, and architectural brands—not a typical local tradesman website.
